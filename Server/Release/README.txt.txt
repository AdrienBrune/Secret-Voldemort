Sous Qt, compiler le code en mode "Release" dans le dossier Release/[Windows/MacOS] selon la cible

Pour Windows :
 - Ouvrir l'invité de commande "Qt 5.15.2" via la recherche windows
 - Taper la commande "windeployqt D:\Workspace\Qt\Projets\Secret-Voldemort-V2.0\SV_SERVER\Release\Windows\release"
 - Executer le fichier releaseWindows.sh pour mettre en place le dossier Release/Windows
Pour MacOS :
 - Executer le fichier releaseMacOS.sh pour générer la release MacOS
